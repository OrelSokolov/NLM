НАСТРОЙКА ДЕЙСТВИЙ КОНТЕКСТНОГО МЕНЮ

Nautilus:

1. Установите пакет nautilus-actions
2. Создайте новое действие для меню.

Пример для загрузки документов в ВК. 1. Action -> Context label: Upload to VK 2. Action -> Icon -> Browse (Выбираем иконку) 3. Command -> Path: python 4. Command -> Parameters: '/home/oleg/python/NerdLibraryManager/network/upload.py' '%f' 5. Execution -> Execution mode -> Display output
